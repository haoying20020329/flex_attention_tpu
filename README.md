# Flex Attention TPU

This repository contains work from a **collaborative course project for EECS 570**.  
The project explores the implementation and experimentation of **FlexAttention kernels on TPU using JAX/Pallas**, including forward and backward attention computation and performance analysis.

## My Contribution (as Hazel Liu)
Hazel Liu contributed to this project as part of the EECS 570 collaboration. Contributions include:

- Implementation and debugging of attention kernel components
- Development and refinement of backward attention computations
- Experimentation and validation of FlexAttention behavior on TPU
- Supporting scripts for performance and kernel analysis
